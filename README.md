# 📘 Palindrome Checker Management System

## 📌 Project Overview

The **Palindrome Checker Management System** is a console-based Java application designed to demonstrate core Java programming fundamentals.

This version implements **Use Case 1 (UC1)** — Application Entry & Welcome Message.

At this stage, the application:

- Starts execution from the `main()` method
- Displays a welcome message
- Shows application version
- Confirms system initialization
- Does NOT yet include palindrome validation logic

---

##


### 🎯 Objective
To establish a clean and structured startup flow for the Palindrome Checker application.

### ✅ Features Implemented
- Java application entry point
- Welcome message display
- Version information display
- System initialization confirmation
